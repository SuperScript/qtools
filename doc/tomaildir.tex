\section{The \cmd{tomaildir} program}

\subsection{Interface}
In \cmd{.qmail}:
\begin{code}%
  | tomaildir \var{dir}
\end{code}
where \cvar{dir} is a Maildir-format directory.

For each email message it processes, \cmd{tomaildir} writes the
message to \cvar{dir}.  If the write succeeds, \cmd{tomaildir} exits
0.  If it encounters a permanent error, \cmd{tomaildir} exits 100.
Otherwise, \cmd{tomaildir} exits 111.
