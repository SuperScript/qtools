\section{The \cmd{822bodyfilter} program}

\subsection{Interface}
\begin{code}%
  822bodyfilter \var{prog}
\end{code}
where \cvar{prog} is one or more arguments specifying a program to
run.

\cmd{822bodyfilter} reads an 822-format email message from standard
input.  It executes \cvar{prog} with the body of the message on
standard input, and prints the message headers followed by the output
from \cvar{prog} to standard output.

If \cmd{822bodyfilter} encounters a temporary error in executing
\cvar{prog}, it exits 111.  If \cmd{822bodyfilter} encounters a
permanent error in executing \cvar{prog}, it exits 100.  Normally,
\cmd{822bodyfilter} yields the same exit code as \cvar{prog}.  The
standard input to \cmd{822bodyfilter} must be seekable.

\cmd{822bodyfilter} considers the end of the message or the first line
that cannot be part of an 822-format header field to terminate the
message headers.  A nonblank line that terminates the headers is part
of the message body.
