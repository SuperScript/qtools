\section{The \cmd{iftoccfrom} program}

\subsection{Interface}
\begin{code}%
  iftoccfrom [ \var{addr} ... ]
\end{code}
where \cvar{addr} is an email address.

\cmd{iftoccfrom} reads an 822-format message from standard input and exits 0 if
\cmd{\$RECIPIENT} is listed in a \cmd{To}, \cmd{Cc}, or \cmd{From}
field, and exits 100 otherwise.  
If one or more \cvar{addr} is present, then \cmd{iftoccfrom} ignores
\cmd{\$RECIPIENT}, and uses each \cvar{addr} instead.

If it encounters a temporary error while reading input, \cmd{iftoccfrom} exits
111.

For example, the \cmd{.qmail} line
\begin{code}%
  | condtomaildir \var{dir} iftoccfrom \var{afriend}
\end{code}
writes correspondence with \cvar{afriend} to \cvar{dir}.
