\section{What is qtools?}
The qtools package comprises several utilities for use with qmail,
typically as part of \cmd{.qmail} command processing.

\cmd{822body} extracts the body from a message.

\cmd{822bodyfilter} filters a message body, preserving the header.

\cmd{822headerfilter} filters a message header, preserving the body.

\cmd{822headerok} tests for a well-formed message header.

\cmd{822fields} extracts field values from a message header.

\cmd{822headers} extracts field copies from a message header.

\cmd{822addr} extracts addresses from a message header.

\cmd{checkaddr} checks an address list for matching addresses.

\cmd{checkdomain} checks an address list for matching domains.

\cmd{iftoccfrom} tests for addresses in if, to, cc, and from header fields.

\cmd{filterto} filters and forwards a message.

\cmd{condtomaildir} conditionally writes a message to a maildir.

\cmd{tomaildir} writes a message to a maildir.

\cmd{replier} is a simple autoreplier.

\cmd{replier-config} configures \cmd{replier}.
