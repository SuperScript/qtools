\section{The \cmd{822fields} program}

\subsection{Interface}
\begin{code}%
  822fields \var{field} ...
\end{code}
where \cvar{field} is an email header name.

\cmd{822fields} reads an 822-format email message from standard input.
For each input header \cvar{field}, \cmd{822fields} prints
one line containing the value of the field to standard output.

\cmd{822fields} considers the end of the message or the first line that
cannot be part of an 822-format header field to terminate the message
message body.

If it encounters an error while processing a message,
\cmd{822fields} exits 111.

If no \cvar{field} appears, \cmd{822fields} looks for \cvar{subject}.
If \cmd{822fields} finds any matching headers, it exits 0.
Otherwise, \cmd{822fields} exits 100.
