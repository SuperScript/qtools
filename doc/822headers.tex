\section{The \cmd{822headers} program}

\subsection{Interface}
\begin{code}%
  822headers \var{field} ...
\end{code}
where \cvar{field} is an email header name.

\cmd{822headers} reads an 822-format email message from standard input.
For each input header \cvar{field}, \cmd{822headers} prints a
full copy of the field, including name, to standard output.

\cmd{822headers} considers the end of the message or the first line that
cannot be part of an 822-format header field to terminate the message
message body.

If it encounters an error while processing a message,
\cmd{822headers} exits 111.

If no \cvar{field} appears, \cmd{822headers} looks for \cvar{subject}.
If \cmd{822headers} finds any matching headers, it exits 0.
Otherwise, \cmd{822headers} exits 100.
